# Car Price Prediction with Machine Learning

## Oasis Infobyte Data Science Internship

### Task 3

## Objective

The objective of this project is to build a machine learning regression model that predicts the selling price of a used car based on features such as brand, car age, fuel type, transmission, and kilometers driven.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook (Google Colab)

---

## Dataset

- Dataset: Vehicle Dataset from CarDekho
- Source: Kaggle

---

## Features Implemented

- Data Loading
- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Feature Engineering
- Brand Extraction
- Car Age Calculation
- Exploratory Data Analysis (EDA)
- One-Hot Encoding
- Correlation Heatmap
- Train-Test Split
- Linear Regression Model
- Random Forest Regressor
- Model Evaluation (MAE, RMSE, R²)
- Feature Importance Analysis

## Visualizations

- Selling Price Distribution

This project demonstrates the complete machine learning workflow for predicting used car prices using regression models. Different regression algorithms are evaluated, and the best-performing model is selected based on performance metrics.
