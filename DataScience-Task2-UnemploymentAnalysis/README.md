# Unemployment Analysis with Python

## Oasis Infobyte Data Science Internship

### Task 2

### Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on unemployment data in India to identify regional and temporal trends. The project also analyzes the impact of the COVID-19 pandemic on unemployment rates using data visualization and statistical analysis.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook (Google Colab)

---

## Dataset

- Dataset: Unemployment in India
- Source: Kaggle

---

## Features Implemented

- Data Loading
- Data Cleaning
- Shape Inspection
- Data Type Analysis
- Null Value Check
- Date Conversion
- Descriptive Statistics
- Region-wise Average Unemployment Analysis
- Month-wise Unemployment Trend
- Time-Series Analysis
- Top 10 States with Highest Average Unemployment Rate
- Correlation Heatmap
- Pre-COVID vs Post-COVID Comparison
- Observations for Each Visualization

---

## Visualizations

- Line Chart
- Bar Chart
- Heatmap

---

## Project Workflow

1. Import Required Libraries
2. Load the Dataset
3. Perform Data Inspection
4. Data Cleaning
5. Exploratory Data Analysis (EDA)
6. Create Visualizations
7. Analyze COVID-19 Impact
8. Draw Conclusions

---

## Conclusion

This project analyzes unemployment trends across different regions of India and examines how unemployment changed before and after the COVID-19 pandemic. Using exploratory data analysis and visualizations, meaningful insights are obtained regarding regional unemployment patterns and labour market changes.
