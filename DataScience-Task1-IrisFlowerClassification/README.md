
# Iris Flower Classification

## Oasis Infobyte Data Science Internship

### Task 1

---

## Objective

Develop a machine learning classification model to predict the species of an Iris flower (Setosa, Versicolor, or Virginica) using its physical measurements.

---

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Dataset

The Iris dataset is loaded directly from the `sklearn.datasets.load_iris()` module. No external dataset download is required.

---

## Project Workflow

- Load the Iris dataset
- Perform Exploratory Data Analysis (EDA)
- Visualize the dataset using Pairplots and Boxplots
- Analyze feature importance through visualization
- Split the dataset into training and testing sets
- Train multiple machine learning models
- Compare model performance
- Evaluate models using Accuracy Score, Confusion Matrix, and Classification Report

---

## Machine Learning Models

- Logistic Regression
- K-Nearest Neighbors (KNN)

---

## Results

- Successfully classified Iris flower species using machine learning.
- Logistic Regression and K-Nearest Neighbors (KNN) models were trained and evaluated.
- KNN achieved the highest classification accuracy on the test dataset.
- Model performance was evaluated using Accuracy Score, Confusion Matrix, and Classification Report.
